# v2026.0 - API Version Removed

This API version has been **permanently removed** and is no longer supported.

## What happened?

API version v2026.0 was a preview version. It was discontinued on **3rd June 2026** and superseded by [v2026.1](/api-docs/v2026.1/). All its documentation has been removed.

## What should I do?

Please migrate to the current supported preview version:

* **[v2026.1](/api-docs/v2026.1/)** - The current supported preview API version

If you need assistance migrating from v2026.0, please refer to the [API versioning strategy](/api-docs/v2026.1/api-versioning-strategy.html) documentation.
